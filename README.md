# TDD in JavaScript
Source files for the course [TDD in JavaScript by Dev Mastery][7c368f77]

  [7c368f77]: https://tddinjs.com "TDD in JavaScript by Dev Mastery"

## Set-up
Using NPM:

    $ git clone https://github.com/dev-mastery/tdd-in-js.git tdd
    $ cd tdd
    $ npm install

## Run Tests
Using NPM:

    $ npm test

for watch mode:

    $ npm test:watch
